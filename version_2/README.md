# 前端全景宇宙：从HTML到AI Web (Version 2)

## 🌌 项目概述

这是一个交互式的前端技术栈可视化项目，以"宇宙构建"为隐喻，展示从基础语言到未来前端技术的完整演化过程。项目旨在让观众在一分钟内直觉地"看懂前端的全貌"。

## ✨ 核心特性

### 🧩 七层技术宇宙
1. **原子层** - HTML/CSS/JavaScript 基础粒子
2. **分子层** - React/Vue/Angular 框架能量核心
3. **系统层** - 工程化工具齿轮管道系统
4. **网络层** - 通信数据脉冲神经网络
5. **应用层** - 状态管理与UI组件架构
6. **表现层** - 性能优化与用户体验光流
7. **前沿层** - WebGPU/Web3/AI前端星门

### 🎬 视觉特效
- **未来科技风格** - Neon Blue + Purple Glow 主题
- **动态粒子系统** - 星空背景与光粒效果
- **Canvas动画** - 框架连接、数据流、网络脉冲、维度空间
- **交互响应** - 鼠标跟随、滚动视差、键盘控制
- **玻璃态设计** - 现代UI与背景模糊效果

### 🎮 交互控制
- **播放控制** - 播放/暂停/重置动画
- **进度指示** - 实时显示探索进度
- **键盘快捷键** - 空格键播放/暂停，R键重置，N键切换旁白
- **滚动触发** - 自动检测层级可见性
- **音频支持** - 背景音乐与旁白系统

## 🛠 技术实现

### 前端技术栈
- **HTML5** - 语义化结构与Canvas元素
- **CSS3** - 高级动画、渐变、滤镜、玻璃态效果
- **JavaScript ES6+** - 模块化、异步处理、Canvas绘图
- **Web APIs** - Intersection Observer、Web Audio、requestAnimationFrame

### 核心功能模块
- **动画引擎** - 基于requestAnimationFrame的流畅动画
- **Canvas绘图** - 多层Canvas实现复杂视觉效果
- **事件系统** - 统一的交互事件处理
- **进度管理** - 智能的动画进度跟踪
- **音频控制** - 背景音乐与旁白同步

## 🚀 使用方法

### 本地运行
1. 克隆或下载项目文件
2. 在项目目录启动HTTP服务器：
   ```bash
   python -m http.server 8000
   ```
3. 打开浏览器访问 `http://localhost:8000`

### 交互操作
- **开始探索** - 点击"开始探索"按钮进入宇宙
- **播放动画** - 点击播放按钮或按空格键
- **暂停/重置** - 使用控制面板按钮或快捷键
- **切换旁白** - 点击旁白按钮或按N键
- **手动导航** - 使用左右箭头键切换层级

### 键盘快捷键
- `空格键` - 播放/暂停动画
- `R` - 重置到开始状态
- `N` - 开启/关闭旁白
- `←/→` - 手动切换层级（暂停状态下）
