# 前端星图：从HTML到未来的Web宇宙

## 项目简介

这是一个交互式的前端技术体系可视化项目，通过动画方式动态展示从 HTML、CSS、JavaScript 到现代框架与工程化工具的演进与关联。项目采用"未来UI演示 + 信息流可视化"的设计风格，具有空间层次感和动静结合的视觉效果。

## 功能特性

### 🎬 六大技术层级展示

1. **HTML层（结构）** - 信息骨架塔
   - 动态展示HTML标签结构
   - 3D悬浮效果和透明几何体
   - 标签元素的浮动动画

2. **CSS层（表现）** - 色彩波纹
   - 渐变色彩波纹扩散效果
   - Flexbox、Grid布局动态切换
   - 响应式布局演示

3. **JavaScript层（逻辑）** - 能量流
   - 粒子系统模拟异步交互
   - Promise、async、DOM关键字展示
   - 动态连线和能量爆发效果

4. **框架宇宙** - 旋转行星
   - React、Vue、Angular三大框架
   - 行星轨道动画
   - 组件网络连接线

5. **工程化系统** - 齿轮机械
   - Webpack、Vite、Babel工具展示
   - 齿轮旋转和数据流动画
   - 构建过程可视化

6. **性能体验层** - 多设备适配
   - 桌面、平板、手机设备展示
   - PWA、WebGPU、Animation技术
   - 自适应界面演示

### 🎮 交互控制

- **开始演示** - 自动播放完整动画序列
- **暂停** - 暂停当前动画
- **重置** - 重置到初始状态
- **滚动触发** - 滚动页面自动激活对应层级
- **鼠标交互** - 悬停效果和光标跟踪

### 🎨 视觉效果

- **科技风格** - 蓝紫色调 + 霓虹光效
- **粒子系统** - 动态粒子和连接线
- **玻璃拟态** - 半透明背景和模糊效果
- **星空背景** - 动态闪烁的星空
- **渐变动画** - 平滑的颜色过渡

## 技术实现

### 前端技术栈
- **HTML5** - 语义化结构
- **CSS3** - 动画、渐变、3D变换
- **JavaScript ES6+** - 交互逻辑和动画控制
- **Canvas API** - 粒子系统和能量流
- **Intersection Observer** - 滚动触发动画
- **Web Animations API** - 高性能动画

### 核心特性
- 响应式设计，支持多设备
- 高性能动画，60fps流畅体验
- 模块化代码结构
- 无依赖纯原生实现

## 使用方法

1. **直接打开**
   ```bash
   # 在浏览器中打开 index.html
   open index.html
   ```

2. **本地服务器**（推荐）
   ```bash
   # 使用 Python 启动本地服务器
   python -m http.server 8000
   
   # 或使用 Node.js
   npx serve .
   ```

3. **访问页面**
   - 打开浏览器访问 `http://localhost:8000`
   - 点击"开始演示"按钮观看完整动画
   - 或滚动页面逐层查看技术展示

## 自定义配置

### 修改动画参数
在 `script.js` 中可以调整：
- 粒子数量：`energyParticles` 数组长度
- 动画速度：各种 `duration` 参数
- 颜色主题：CSS 变量和渐变色

### 添加新的技术层级
1. 在 HTML 中添加新的 `.layer` 元素
2. 在 CSS 中定义对应样式
3. 在 JavaScript 中实现动画逻辑
